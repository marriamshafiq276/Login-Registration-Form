# Login & Registration Form

A simple responsive login and registration page built with HTML, CSS, and JavaScript.

## Overview

This project demonstrates a clean form UI with a toggle effect between a login panel and a registration panel. The design uses modern styling and animated transitions to switch between the two views.

## Features

- Login form with username and password fields
- Registration form with username, email, and password fields
- Toggle animation between login and registration panels
- Social login icons for visual styling
- Responsive centered layout with gradient background

## Files

- `index.html` - Main page structure and form markup
- `style.css` - Styling and animation for the login/register UI
- `script.js` - Toggle logic for switching between forms

## How to Use

1. Open `index.html` in a web browser.
2. Click the `Register` button to reveal the registration panel.
3. Click the `Login` button to return to the login panel.

## Notes

- The forms are visual/demo-only and do not submit to a backend.
- Social icons use the Boxicons CDN for styling.
- You can customize the form fields, colors, and animation timings in `style.css`.
